# Emotion-Sensing-Robot
An intelligent emotion detection system recognizing human facial expressions in real-time using MobilenetV2+FER2013 model. Built with Python, TensorFlow 2.12.0, Keras 2.12.0, OpenCV 4.8.0.74, NumPy 1.23.5, Pandas, and h5py 3.8.0. The system captures facial features, processes them through deep learning, and classifies emotions accurately
